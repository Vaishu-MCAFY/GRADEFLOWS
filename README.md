# GRADEFLOWS
Academic performance analytics portal GradeFlow is a web-based academic performance management system developed using Python, Django, MongoDB, HTML, CSS, and JavaScript. It helps faculty members manage student records, upload marks through CSV files, generate reports, and analyze academic performance using dashboards.
